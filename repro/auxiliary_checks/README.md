# Auxiliary Checks

This directory stores small, tracked summaries for the post-canonical checks referenced in the manuscript.

Subdirectories:
- `u27_eta05_retraining/`: eta-aligned retraining validation and held-out summaries
- `u36_replication/`: second-universe validation and held-out summaries
- `cctalibp/`: cost-calibrated linear-convex comparator selection and held-out summaries

The large raw experiment trees remain under `prl-dow30/outputs/` locally; this directory keeps only the compact files needed to inspect the paper-facing numbers after a fresh clone.
