# Cost Sweep Captions v1

## Figure C1. Executed-Path Selected-Point Advantage Across Cost Regimes

This figure plots the paired-median executed-path delta Sharpe for the locked comparison `eta=0.5` minus `eta=1.0` across the denser cost grid. It matters because the zero-cost row remains near-flat while the positive-cost rows show a larger realized-path advantage, reinforcing the same friction-sensitive interpretation as the main selected-point result without turning the sweep into a new main finding.

## Figure C2. Diagnostic Gaps Across Cost Regimes

This figure plots two diagnostic quantities across the same cost grid: the final executed-versus-target path gap at `eta=0.5` and the target-minus-executed turnover gap at `eta=0.5`. It matters because the target and executed paths remain separated once frictions intervene, but the pattern should be read narrowly: the turnover gap is nearly flat, the path gap becomes more visible in the positive-cost region, and the series is not strictly monotone.
